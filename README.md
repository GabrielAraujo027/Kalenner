# Kalenner

Kalenner é um projeto (Frontend + Backend) para gerenciar calendários e eventos. Este repositório contém o frontend, o backend e materiais de demonstração.

## Conteúdo do repositório (visão geral)
- Backend/ — código do servidor (API)
  - Kalenner.sln
  - Migrations/ (Entity Framework)
  - Controllers/, Entities/, Services/, Infra/
- Frontend/ — código do cliente (interface)
  - client/, server/, shared/ (estrutura do app)
  - components.json
  - .prettierrc, .prettierignore, .gitignore, .gitkeep
- Video_funcionalidades.mp4 — vídeo curto demonstrando algumas funcionalidades

## Funcionalidades
- Criar, editar e remover eventos no calendário
- Visualização por dia/semana/mês
- Autenticação de usuários (se aplicável)
- Notificações e lembretes (se aplicável)

A lista acima deve ser ajustada conforme as funcionalidades reais implementadas.

## Algumas Tecnologias
- Backend: .NET, Entity Framework, Identity.
- Frontend: React.
- Banco de dados: A principio desenvolvi com SQL Server mas pode ser alterado conforme necessidade.
